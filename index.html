My Work

ChatGPT AI Clone

I created an AI chatbot using LlamaIndex and GPTIndex, integrating custom knowledge bases to provide personalized interactions. This chatbot utilizes GPT-3's language model in Python to deliver human-like responses.

BSwipe Web Application

BSwipe serves as an innovative platform designed to facilitate the buying and selling of dining hall meal swipes at UCLA. Our robust system allows users to create and login as their unique user so that they can browse and create swipe request posts. Additionally, users can keep track of their transaction history through their personalized user profile. We leveraged the MERN technical stack to develop and deploy a fully functional website, featuring dynamic data processing, efficient client-to-backend data uploading, and server-side data search capabilities (MongoDB, Express.js, React, Node.js)

Line Following Car

I designed and constructed an Arduino-based TI-RSLK robotic car for line-following tasks. I developed an Arduino program, using C and C++, that utilizes a PID controller, sensor data, wheel encoders, and other techniques to enable the robotic car to accurately follow a black line on a track and successfully complete the course within a 15-second timeframe, demonstrating precise navigation even along curved sections of the track.
